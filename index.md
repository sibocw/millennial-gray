# Hello,

Millennial Gray is a [Jekyll](https://jekyllrb.com/)-based personal website template with boring but elegant millennial gray aesthetics, deliberately made to not stand out. It allows you to write your content in Markdown; HTML code is generated automatically. Visit its [GitHub repository](http://sibocw.github.io/millennial-gray) to see how to set it up.
